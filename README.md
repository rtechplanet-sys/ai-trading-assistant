# AI Trading Platform

An open-source platform for algorithmic trading powered by Artificial Intelligence.

## Overview

This project provides a modular framework for building AI-driven trading strategies across multiple financial markets.

## Supported Markets

- Forex
- Stocks
- Cryptocurrencies
- Commodities
- Indices
- ETFs

## Features

- AI-assisted strategy generation
- Technical indicators
- Risk management
- Backtesting
- Paper Trading
- Live Trading
- Multi-broker support
- Performance analytics

## Technologies

- Python
- FastAPI
- Pandas
- NumPy
- PyTorch
- PostgreSQL
- Docker
